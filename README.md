《网络守护者》是一款融合**网络安全概念**的塔防策略游戏。玩家扮演网络安全工程师，通过部署防火墙、杀毒软件、加密节点等防御设施，抵御病毒、蠕虫、黑客、DDoS 攻击等网络威胁，守护核心服务器的安全。

---

## 游戏特色

| 特性 | 说明 |
|------|------|
| 网络安全主题 | 防御塔和敌人都基于真实网络概念（防火墙、杀毒软件、DDoS、Rootkit 等） |
| 6 大关卡 | 从入门网络到终极防线，难度递增，地图各异 |
| 10 种敌人 | 病毒、蠕虫、木马、勒索软件、黑客、间谍软件、Rootkit、僵尸网络、DDoS、APT |
| 激怒机制 | 敌人受击 4 次后进入激怒状态，获得加速、减伤和回血 |
| 金币倍率 | 每关金币获取倍率不同，后期关卡资源紧张，考验策略 |
| 概率掉落 | 击杀敌人有概率掉落金币，增加不确定性 |
| 音效系统 | 10 种程序化音效（射击、击中、爆炸、金币、胜利等） |

## 安装与运行

### 环境要求
- Python 3.10+
- pygame 2.6+

### 安装依赖
```bash
pip install pygame
```

### 运行游戏
```bash
python main.py
```

## 项目结构

```
network_defender/
├── main.py              # 游戏入口
├── config.py            # 游戏配置（颜色、防御塔、敌人、关卡）
├── game_engine.py       # 游戏引擎（主循环、状态管理）
├── map_system.py        # 地图系统（路径生成、网格）
├── tower_system.py      # 防御塔系统（攻击、升级、投射物）
├── enemy_system.py      # 敌人系统（移动、受击、激怒、波次）
├── ui_system.py         # UI 系统（菜单、游戏界面、结算）
├── effect_system.py     # 特效系统（粒子、伤害数字、爆炸）
├── sound_system.py      # 音效系统（程序化生成音效）
├── 课设报告.md          # 课程设计报告
├── 打包说明.md          # Windows 打包指南
├── build.bat            # 一键打包脚本
└── README.md            # 项目说明
```

---

## 打包为可执行文件

在 Windows 上运行以下命令，生成独立的 `.exe` 文件：

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name NetworkDefender main.py
```

或使用一键脚本：
```bash
build.bat
```

打包完成后，`dist/NetworkDefender.exe` 可直接双击运行，无需 Python 环境。
