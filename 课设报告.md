# Python网络自动化课程设计报告

## 网络守护者 (Network Defender)

---

## 一、项目概述

### 1.1 项目背景
随着网络技术的快速发展，网络安全问题日益突出。本项目将网络自动化概念与游戏设计相结合，开发了一款名为《网络守护者》的塔防游戏。玩家在游戏中扮演网络安全工程师，通过部署各种网络防御设施来保护核心服务器免受病毒、黑客和恶意软件的攻击。

### 1.2 开发环境
- **开发工具**: PyCharm 2024.1.6
- **编程语言**: Python 3.13.5
- **游戏库**: Pygame 2.6.0
- **操作系统**: Windows 10/11 / Linux / macOS

### 1.3 项目特点
1. **主题创新**: 将网络自动化概念融入游戏设计
2. **操作简单**: 鼠标点击即可操作，支持快捷键
3. **老少皆宜**: 卡通风格画面，难度循序渐进
4. **教育意义**: 在游戏中学习网络安全基础知识

## 二、需求分析

### 2.1 功能需求
1. **地图系统**: 网格化地图，包含路径、基地、出生点

## 三、系统设计

### 3.1 架构设计
项目采用模块化设计，各系统职责明确：

```
network_defender/
├── main.py              # 游戏入口
├── config.py            # 游戏配置（常量、颜色、属性）
├── game_engine.py       # 游戏引擎（主循环、状态管理）
├── map_system.py        # 地图系统（网格、路径、渲染）
├── tower_system.py      # 防御塔系统（攻击、升级、投射物）
├── enemy_system.py      # 敌人系统（移动、伤害、波次）
├── effect_system.py     # 特效系统（粒子、动画、震动）
├── ui_system.py         # UI系统（按钮、面板、菜单）
└── requirements.txt     # 依赖文件
```

### 3.2 核心类设计

#### 3.2.1 Game类（游戏引擎）
- **职责**: 游戏主循环、事件处理、状态管理
- **核心方法**:
  - `init_game()`: 初始化游戏数据
  - `run()`: 主循环
  - `_handle_events()`: 事件处理
  - `_update(dt)`: 逻辑更新
  - `_draw()`: 画面渲染

#### 3.2.2 GameMap类（地图系统）
- **职责**: 地图生成、网格管理、坐标转换
- **核心方法**:
  - `_generate_path()`: 生成敌人路径
  - `is_valid_tower_position()`: 检查有效位置
  - `place_tower()`: 放置防御塔

#### 3.2.3 Tower类（防御塔）
- **职责**: 攻击逻辑、目标寻找、升级系统
- **核心方法**:
  - `find_target()`: 寻找目标
  - `fire()`: 发射投射物
  - `upgrade()`: 升级防御塔

#### 3.2.4 Enemy类（敌人）
- **职责**: 路径移动、受到伤害、状态效果
- **核心方法**:
  - `update(dt)`: 更新位置和状态
  - `take_damage()`: 受到伤害
  - `apply_slow()`: 应用减速效果

#### 3.2.5 WaveManager类（波次管理）
- **职责**: 敌人生成、波次控制、奖励计算
- **核心方法**:
  - `start_wave()`: 开始新波次
  - `update(dt)`: 更新波次状态
  - `is_wave_ready()`: 检查是否可以开始下一波

## 五、关键技术实现

### 5.1 路径跟随算法
敌人沿预定义路径点移动，使用向量计算方向：

```python
def update(self, dt):
    if self.path_index < len(self.path) - 1:
        target_x, target_y = self.path[self.path_index + 1]
        dx = target_x - self.x
        dy = target_y - self.y
        distance = math.sqrt(dx * dx + dy * dy)
        
        if distance <= current_speed:
            self.x = target_x
            self.y = target_y
            self.path_index += 1
        else:
            self.x += (dx / distance) * current_speed
            self.y += (dy / distance) * current_speed
```

### 5.2 目标寻找算法
防御塔自动寻找范围内最近的敌人：

```python
def find_target(self, enemies):
    closest_enemy = None
    closest_distance = float('inf')
    
    for enemy in enemies:
        distance = math.sqrt((enemy.x - self.x) ** 2 + (enemy.y - self.y) ** 2)
        if distance <= self.range and distance < closest_distance:
            closest_distance = distance
            closest_enemy = enemy
    
    return closest_enemy
```

### 5.3 粒子特效系统
使用面向对象方式管理粒子生命周期：

```python
class Particle:
    def update(self, dt):
        self.x += self.vx * dt
        self.y += self.vy * dt
        self.lifetime -= dt * 1000
        if self.lifetime <= 0:
            self.alive = False
```
