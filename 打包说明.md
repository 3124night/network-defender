# 网络守护者 - Windows 打包指南

## 环境要求

- Windows 10/11 系统
- Python 3.10+（已安装）
- pygame 已安装

## 打包步骤

### 1. 安装 PyInstaller

打开 CMD 或 PowerShell，执行：

```bash
pip install pyinstaller
```

### 2. 进入游戏目录

```bash
cd network_defender
```

### 3. 执行打包命令

```bash
pyinstaller --onefile --windowed --name 网络守护者 main.py
```

参数说明：
- `--onefile`：打包成单个 exe 文件
- `--windowed`：不显示控制台窗口
- `--name`：生成的 exe 文件名

### 4. 获取可执行文件

打包完成后，在 `dist/` 目录下找到：

```
dist/
  └── 网络守护者.exe
```

### 5. 分发文件

将 `网络守护者.exe` 发送给小组成员，对方直接双击即可运行，**不需要安装 Python**。

---

## 常见问题

### 打包后文件很大？

正常现象。单文件 exe 会包含 Python 解释器和所有依赖，通常 15-30MB。

### 杀毒软件报毒？

PyInstaller 打包的程序可能被误报。将文件添加到杀毒软件白名单即可。

### 运行时闪退？

尝试去掉 `--windowed` 重新打包，这样能看到错误信息：

```bash
pyinstaller --onefile --name 网络守护者 main.py
```

---

## 快捷脚本

也可以创建一个 `build.bat` 文件，内容如下：

```bat
@echo off
echo 正在打包 网络守护者...
pyinstaller --onefile --windowed --name 网络守护者 main.py
echo 打包完成！文件位于 dist/网络守护者.exe
pause
```

双击运行 `build.bat` 即可自动打包。
